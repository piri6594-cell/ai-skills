# 일일 리뷰 시스템 규칙

## 리뷰 출력 형식
- 리뷰는 마크다운 파일로만 저장 (채팅에 전체 내용 나열 금지)
- 파일명 패턴:
  - Mode A: `YYYYMMDD_review.md`
  - Mode B: `YYYYMMDD_learning.md`
  - 아침: `YYYYMMDD_morning.md`
- 저장 위치: `OneDrive/바탕 화면/0. AI 에이전트/daily-reviews/`

## SESSION_LOG 연동
- 18:30 리뷰 시 `SESSION_LOG.md`의 당일 엔트리를 수집하여 리뷰에 포함
- `session_close.py --scan` 결과에서 승격 추천이 있으면 리뷰 보고서에 명시
- 5세션 이상 축적 시 "패턴 리뷰" 섹션을 보고서에 추가

## 스킬 업데이트 판단 기준
- 같은 스킬에서 수정 요청 2회 이상 → SKILL.md 업데이트 플래그
- 사용자가 같은 지시를 3개 이상 세션에서 반복 → 새 글로벌 규칙 추가
- 스킬 트리거 키워드 감지되었으나 실제 스킬 미사용 → 트리거 매핑 점검
- 7일 이상 사용 안 된 스킬 → stale 플래그 (삭제는 아님)

## 에이전트 평가 기준
- Claude 사수: 판단 개입 횟수, 규칙 업데이트 기여, 에러 사전 방지
- Codex 조수: 결과물 생성 수, 검증 통과율, 실행 성공률
- 점수: 0~10 (0=미사용, 5=기본 수행, 8=우수, 10=완벽)

## Mode B 검색 추천 필터
- 적용 가능성 "높음": 현재 스킬/워크플로우에 즉시 적용 가능
- 적용 가능성 "중간": 약간의 수정/학습 필요
- 적용 가능성 "낮음": 참고용, 당장 필요 없음
- 추천 우선순위: 높음 > 중간 (낮음은 참고로만 기재)
