# 토목 실시설계 수량산출서 작성 가이드

## 핵심 안전 원칙
- **원본 파일 절대 수정 금지** — 반드시 복사본에 작업하고 새 파일명으로 저장
- 편집 전 대상 시트·수식·병합 셀·반복 행 패턴을 먼저 파악
- 대량 입력 전 2~3개 샘플 행 검증 필수
- `#REF!`, `#DIV/0!` 등 깨진 참조 먼저 수리 후 수량 입력
- 완성 파일이 Excel에서 오류 없이 열리는 것을 확인 후 종료

## 법적 근거 및 적용기준
- 건설공사 표준품셈 (국토교통부 고시, 매년 개정)
- 건설공사 표준시방서 (KCS 시리즈)
- 건설기술진흥법 제44조 (설계도서 작성기준)
- 국가계약법 시행규칙 제7조 (원가계산 기준)
- 해당 공종별 KDS 설계기준 적용

## 작업 순서 (SOP)

### Step 1 — 입력 정보 확인
사용자에게 다음을 확인한다:
- 공사 이름 (예: ○○소하천 정비공사)
- 적용 공종 목록 (토공 / 콘크리트공 / 철근공 / 거푸집공 / 비탈면보호공 / 배수공 / 포장공 등)
- 설계도면 치수 또는 수치 데이터 (사용자 제공 또는 입력)
- 단위 체계 (m, m², m³, ton, EA 등)
- 기존 파일 수정인지 신규 작성인지 확인

### Step 2 — 파일 준비
- 기존 파일이 있으면 **복사본 생성 후 작업** (원본 보존)
- 신규 작성 시 `xlsx` 스킬을 함께 사용하여 아래 시트 구조로 작성:

| 시트명 | 내용 |
|--------|------|
| 표지 | 공사명, 발주처, 설계자, 작성일 |
| 총괄수량표 | 공종별 수량 집계 (SUM 수식 연동) |
| 공종별 산출근거 | 각 공종 상세 계산식 (1개 공종 = 1섹션) |

### Step 3 — 공종별 수량 계산 원칙
- **도면 참조**: 어느 도면의 어느 부재를 산출하는지 명시 (예: "구조도 S-1, 기초 슬래브")
- **계산식**: Excel 수식 사용 필수 — Python 계산값 하드코딩 금지
- **할증률**: 콘크리트 1.02, 철근 1.03, 토사 1.20~1.30 (L값) 등 표준품셈 기준 적용
- **단위 합계**: 소계 → 중계 → 합계 구조

### Step 4 — 공종별 검증 체크리스트

#### 토공
- [ ] 터파기 = 구조물 외면 + 여유폭(0.5m 이상)
- [ ] 되메우기 = 터파기량 − 구조물 체적
- [ ] 잔토처리 = 되메우기 후 잉여토 (토량변화율 C값 적용)
- [ ] 토량변화율 (L값, C값) 표준품셈 기준 적용

#### 콘크리트공
- [ ] 설계기준강도(fck) 명기 (18, 21, 24, 27 MPa 등)
- [ ] 부재별 체적 계산 후 할증(+2%) 적용
- [ ] 레미콘 규격 명시 (25-21-150 등)

#### 철근공
- [ ] 주철근 + 배력철근 + 스터럽 구분 산출
- [ ] 이음길이 포함 (KDS 14 20 52: 40d 이상)
- [ ] 단위중량 적용 (KS D 3504: HD10=0.617kg/m, HD13=0.995kg/m 등)
- [ ] 할증 +3% 적용

#### 거푸집공
- [ ] 접촉면적만 산출 (콘크리트 노출면)
- [ ] 전용횟수에 따른 규격 구분 (합판거푸집, 유로폼 등)

#### 비탈면보호공
- [ ] 비탈면 경사 명기 (1:0.5, 1:1.0, 1:1.5 등)
- [ ] 사면 면적 = 수직높이 × √(1+경사²) × 연장

#### 배수공 (암거, 측구)
- [ ] 관경 및 재질 명기 (원심력 철근콘크리트관, PE파형관 등)
- [ ] 연장 산출 (도심선 기준)

### Step 5 — 서식 기준
- **폰트**: 맑은 고딕 (본문 10pt, 표제 12pt Bold)
- **색상 규칙**:
  - 파란 텍스트 = 사용자 입력값 (수치)
  - 검정 텍스트 = Excel 수식
  - 노란 배경 = 검토 필요 셀
- **숫자 형식**: 소수점 둘째 자리, 천 단위 콤마
- **단위**: 열 헤더에 반드시 단위 표기 (m, m², m³, ton, EA)

### Step 6 — 최종 검증
- #REF!, #DIV/0! 등 수식 오류 0건 확인
- 총괄수량표 합계 vs 각 공종별 합계 교차검증
- Excel에서 오류 없이 열리는지 확인 후 납품
